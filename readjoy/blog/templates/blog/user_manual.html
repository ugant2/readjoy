Welcome to user manual

Welcome to the Travel Blog User Manual where you can read the awesome
travel stories and experience shared by the creative people over Nepal and beyond.
This User Manual will provide steps by steps to guide to explore this awesome Travel Blog.
It will help you to use all the features and guide you to solve issues that might be faced by you during the time.

Guest User

Since it is web based system, guest user can read the blog and contact administrator without any obstacles.

Register

  • Click on Register menu in the navigation bar.
  • Fill in Username and Password. Its quite easy though, later you can add further information on your profile page.
  • Now click on Register Button to register as user.

Logging In

  • Go to the navigation bar and click on Login to get access into blog.
  • Type Username and Password to login into blog.
  • Click on Login button to login into blog.

Logging Out

  • To logout from the blog simply click on logout in the navigation bar.

Writing Post

  • Go to View Profile in the navigation bar.
  • Click on Edit button.
  • Make up the necessary change.

Contacting Administration

  • Click on Contact on navigation bar.
  • Fill email address, subject, and e-mail body.
  • Click on Send button to send e-mail.

Administration

For administrations of the Blog, s/he can perform various task and have lots of privileges in contrast to other normal users.
Lets navigate ho to use administration site in the blog.

To Enter into Administration Site

  • Go to address bar of the web browser and type www.travelblog.com/admin.
  • Login window will pop up. Fill in the username and password.
  • Click on login button.

Administration Logging Out

  • To logout click on logout in the top-right corner of the blog.

Adding User

  • To add user, click on add of User column.
  • Fill necessary information.
  • Click on save.

To change Post, Comment, and Profile

Admin has right ot make any changes like deleting, updating on the profile, post, and comment of the user. Here, is steps for Posts for which Comment and Profile follows the same step.

  • 1. Click on Add on the Profile column.
  • 2. Fill all the information regarding user.
  • 3. Click on save.

To view all the User

  • Click on Profile

To select Individual User

  • Mark the interested User.

To Delete User

  • Click Action, Select Delete, and Click GO.

To change Password

  • 1. Go to admin dashboard, click on change password on top-right of the blog.
  • 2. Fill necessary information.
  • Click on change
